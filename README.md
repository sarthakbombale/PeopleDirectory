# People Directory App

A dynamic and interactive **People Directory** application built with **React**. This app allows users to manage a list of people with full CRUD operations and demonstrates more advanced React patterns such as component composition, state management with hooks, and reusable form components.

---

## Features

- **Add Person**: Users can add new entries with name, email, and other details.
- **Edit Person**: Modify existing entries directly in the list.
- **Delete Person**: Remove any person from the directory.
- **Search & Filter**: Quickly find people in the directory.
- **Dynamic Validation**: Form input validation for better UX.
- **Responsive Design**: Works well on desktop and mobile devices.

---

## Tech Stack

- **Frontend:** React 19  
- **State Management:** React `useState` and `useReducer` for local state  
- **Styling:** React Bootstrap 5  
- **Build Tool:** Vite
